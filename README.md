# 📊 Python-data-science-libraries-Notebooks

This repository contains my hands-on practice with **Python data science libraries**, focusing on **data analysis, visualization, and exploratory data analysis (EDA)**. It showcases step-by-step work with real datasets and demonstrates core skills used in data science workflows.

---

## 🛠️ Tools & Libraries
- **Python** – main programming language  
- **NumPy** – numerical computations, arrays, and statistical operations  
- **Pandas** – data manipulation, cleaning, and grouping  
- **Matplotlib & Seaborn** – data visualization (line plots, bar charts, histograms, heatmaps)  
- **Jupyter Notebook** – interactive environment for coding and visual exploration  

---

## 🔍 Skills Practiced
- **NumPy:** Arrays, indexing, mathematical and statistical functions  
- **Pandas:** Data cleaning, filtering, grouping, pivot tables, handling missing data  
- **Matplotlib & Seaborn:** Line plots, bar charts, scatter plots, heatmaps, trend visualization  
- **EDA Techniques:** Identifying trends, outliers, extreme values, exposure analysis, risk scoring  

---

## 📈 Key Learnings
- How to manipulate and clean data for analysis  
- How to visualize patterns and trends effectively  
- How to perform **risk analysis, variance analysis, and ranking**  
- How to translate raw datasets into **actionable insights**
